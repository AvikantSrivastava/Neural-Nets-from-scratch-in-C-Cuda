# Neural Nets from scratch in C++/Cuda

## Compile and Run

### Using NVCC [Cuda]
```bash
❯ nvcc -std=c++17 test.cu -o testfile          
❯ ./testfile              
```

### Using G++ [C++]
```bash
❯ g++ -std=c++17 test.cpp -o testfile          
❯ ./testfile              
```
